React Weather App

A 5-day weather forecast application built with React JSX, HTML and CSS. The user can search for any city and the search triggers an Ajax call to the Open Weather API which returns formatted data in the form of a JSON file. The app then collects the weather data, stores it internally and displays it through the UI components. By applying the React Hook, useState, the user is able to switch between °F & °C temperatures. Lastly, by using Geolocation, we allow the user to see the weather conditions in their current location.
